### Requirements

* Ansible
* Make

### Как запустить приложение

**Клонируйте репозиторий**

```bash
git clone git@github.com:despot84/devops-for-developers-project-76/.git
```
**Выполнить**

```bash
make install-role

make redmine-deploy
```
